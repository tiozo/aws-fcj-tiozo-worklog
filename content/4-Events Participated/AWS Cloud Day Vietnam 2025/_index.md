# AWS Cloud Day Vietnam 2025

## Event Details
- **Event name:** AWS Cloud Day Vietnam 2025
- **Date and time:** October 2025
- **Location:** Bitexco Financial Tower
- **Role:** Participant

## Event Description
AWS Cloud Day Vietnam 2025 is a major annual conference showcasing the latest AWS cloud technologies and industry best practices. The event featured multiple tracks covering various aspects of cloud computing, with a focus on migration, modernization, and AI integration.

## Sessions Attended

### Track 4: Migration & Modernization

#### 1:15 PM – 1:30 PM | Opening
- **Content:** Event opening and welcome session

#### 1:30 PM – 2:00 PM | Completing a large-scale migration and modernisation with AWS
- **Speaker:** Hieu Hoang, Senior Manager, Solutions Architect, AWS
- **Content:** Strategic approaches and best practices for executing large-scale cloud migration and modernization projects

#### 2:00 PM – 2:30 PM | Modernizing Applications with Generative AI-Powered Tools
- **Speakers:** Chinh Hoang Minh, Senior Solutions Architect, AWS; Harry Nguyen, Director of Engineering, Techcombank
- **Content:** Leveraging generative AI tools to accelerate application modernization processes

#### 2:30 PM – 3:00 PM | Redefining Modernization: Integrating Generative AI into the DevOps Lifecycle on AWS
- **Speakers:** Quan Phuong, Solutions Architect, AWS; Le Hai Duy, Solution Expert, VPBank; Nguyen Thai Binh, Head of DevOps, Renova Cloud
- **Content:** Exploring how organizations drive modernization by integrating Gen AI into AWS DevOps lifecycle. Discussion highlighted AI's potential to boost efficiency, accelerate development & enhance software delivery quality. Covered practical applications such as automated code generation, intelligent testing, proactive monitoring & optimized resource management on AWS.

#### 3:00 PM – 3:30 PM | Break
- **Content:** Networking break

#### 3:30 PM – 4:15 PM | Panel Discussion: Application Modernization: Accelerating Business Transformation
- **Moderator:** Hieu Hoang, Senior Manager, Solutions Architect, AWS
- **Panelists:**
  - Gia Hieu Dinh, CIO, F88
  - Harry Nguyen, Director of Engineering, Techcombank
  - Vincent Nguyen, Managing Director, Nam Long Group
- **Content:** Industry leaders sharing insights on how application modernization accelerates business transformation

#### 4:15 PM – 4:45 PM | Transforming VMware with AI-driven cloud modernisation
- **Moderator:** Trung Dang, Senior Solution Architect, AWS
- **Content:** Strategies for modernizing VMware environments using AI-driven cloud solutions

#### 4:45 PM – 5:15 PM | AWS security at scale: From development to production
- **Moderator:** Huy Tran, Senior Solutions Architect, AWS
- **Content:** Implementing comprehensive security practices across the entire development lifecycle on AWS
