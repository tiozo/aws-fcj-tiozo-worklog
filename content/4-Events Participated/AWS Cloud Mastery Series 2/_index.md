# AWS Cloud Mastery Series #2 - DevOps on AWS

## Event Details
- **Event name:** AWS Cloud Mastery Series #2
- **Date and time:** Monday, November 17, 2025 | 8:30 AM – 5:00 PM GMT+7
- **Location:** Bitexco Financial Tower
- **Role:** Participant

## Event Description
Full-day intensive workshop focused on DevOps practices and AWS services for continuous integration, continuous deployment, and infrastructure automation.

## Sessions Attended

### Morning Session (8:30 AM – 12:00 PM)

#### 1. Welcome & DevOps Mindset
- **Time:** 8:30 – 9:00 AM
- **Content:** 
  - Recap of AI/ML session
  - DevOps culture and principles
  - Benefits and key metrics (DORA, MTTR, deployment frequency)

#### 2. AWS DevOps Services – CI/CD Pipeline
- **Time:** 9:00 – 10:30 AM
- **Content:**
  - **Source Control:** AWS CodeCommit, Git strategies (GitFlow, Trunk-based)
  - **Build & Test:** CodeBuild configuration, testing pipelines
  - **Deployment:** CodeDeploy with Blue/Green, Canary, and Rolling updates
  - **Orchestration:** CodePipeline automation
  - **Demo:** Full CI/CD pipeline walkthrough

#### 3. Infrastructure as Code (IaC)
- **Time:** 10:45 AM – 12:00 PM
- **Content:**
  - **AWS CloudFormation:** Templates, stacks, and drift detection
  - **AWS CDK:** Constructs, reusable patterns, and language support
  - **Demo:** Deploying with CloudFormation and CDK
  - **Discussion:** Choosing between IaC tools

### Afternoon Session (1:00 – 5:00 PM)

#### 4. Container Services on AWS
- **Time:** 1:00 – 2:30 PM
- **Content:**
  - **Docker Fundamentals:** Microservices and containerization
  - **Amazon ECR:** Image storage, scanning, lifecycle policies
  - **Amazon ECS & EKS:** Deployment strategies, scaling, and orchestration
  - **AWS App Runner:** Simplified container deployment
  - **Demo & Case Study:** Microservices deployment comparison

#### 5. Monitoring & Observability
- **Time:** 2:45 – 4:00 PM
- **Content:**
  - **CloudWatch:** Metrics, logs, alarms, and dashboards
  - **AWS X-Ray:** Distributed tracing and performance insights
  - **Demo:** Full-stack observability setup
  - **Best Practices:** Alerting, dashboards, and on-call processes

#### 6. DevOps Best Practices & Case Studies
- **Time:** 4:00 – 4:45 PM
- **Content:**
  - Deployment strategies: Feature flags, A/B testing
  - Automated testing and CI/CD integration
  - Incident management and postmortems
  - Case Studies: Startups and enterprise DevOps transformations

#### 7. Q&A & Wrap-up
- **Time:** 4:45 – 5:00 PM
- **Content:**
  - DevOps career pathways
  - AWS certification roadmap

## Outcomes and Value Gained

### Technical Skills
- **CI/CD Mastery:** Hands-on experience with AWS CodeCommit, CodeBuild, CodeDeploy, and CodePipeline
- **Infrastructure as Code:** Practical knowledge of CloudFormation and CDK for infrastructure automation
- **Container Orchestration:** Understanding of ECS, EKS, and App Runner deployment strategies

### DevOps Practices
- **Deployment Strategies:** Blue/Green, Canary, and Rolling deployment techniques
- **Monitoring & Observability:** CloudWatch and X-Ray implementation for full-stack visibility
- **Best Practices:** Industry-standard approaches to testing, incident management, and postmortems

### Professional Development
- **DevOps Culture:** Deep understanding of DevOps principles and cultural transformation
- **Career Pathways:** Clear guidance on DevOps career progression and AWS certification roadmap
- **Real-world Applications:** Case studies from startups and enterprise DevOps transformations

This comprehensive workshop provided essential DevOps skills and AWS service expertise for modern cloud-native development and operations.
