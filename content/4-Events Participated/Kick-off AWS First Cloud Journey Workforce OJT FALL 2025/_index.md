# Kick-off AWS First Cloud Journey Workforce OJT FALL 2025

## Event Details
- **Event name:** Kick-off AWS First Cloud Journey Workforce OJT FALL 2025
- **Date and time:** September 2025
- **Location:** Bitexco Financial Tower
- **Role:** Participant

## Event Description
The official kick-off event for AWS First Cloud Journey Workforce On-the-Job Training (OJT) program for Fall 2025. This event featured keynote presentations and industry sharing sessions focused on AWS cloud journey direction and career development in DevOps.

## Sessions Attended

### Keynote & Industry Sharing

#### 1. AWS First Cloud Journey & Future Direction
- **Time:** 9:15 – 9:40
- **Speaker:** Nguyễn Gia Hưng – Head of Solutions Architect, AWS Vietnam
- **Content:** Overview of AWS First Cloud Journey program and strategic direction for the future

#### 2. DevOps & Future Career
- **Time:** 9:40 – 10:05
- **Speaker:** Đỗ Huy Thắng – DevOps Lead, VNG
- **Content:** DevOps career development and future opportunities in the industry

## Outcomes and Value Gained

### Program Understanding
- **AWS FCJ Program:** Comprehensive understanding of the AWS First Cloud Journey Workforce OJT program structure and objectives
- **Career Pathways:** Clear insights into cloud computing career development and progression opportunities

### Industry Insights
- **AWS Strategy:** Direct insights from AWS Vietnam leadership on cloud adoption trends and future directions
- **DevOps Career:** Professional guidance on DevOps career development from industry expert at VNG

### Professional Development
- **Networking:** Connected with AWS leadership and industry professionals
- **Program Alignment:** Better understanding of how to maximize the OJT program benefits
- **Industry Trends:** Exposure to current and future trends in cloud computing and DevOps

This kick-off event provided essential foundation knowledge for the AWS First Cloud Journey Workforce OJT program and valuable career guidance from industry leaders.
