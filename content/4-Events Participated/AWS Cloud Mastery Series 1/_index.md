# AWS Cloud Mastery Series #1

## Event Details
- **Event name:** AWS Cloud Mastery Series #1
- **Date and time:** Saturday, November 15, 8:00 AM - 11:30 AM GMT+7
- **Location:** Bitexco Financial Tower
- **Role:** Participant

## Event Description
The first session of AWS Cloud Mastery Series focusing on AI/ML services and Generative AI capabilities. This hands-on workshop provided comprehensive coverage of AWS AI/ML landscape, Amazon SageMaker platform, and Amazon Bedrock for generative AI applications.

## Sessions Attended

### 8:30 – 9:00 AM | Welcome & Introduction
- **Content:** 
  - Participant registration and networking
  - Workshop overview and learning objectives
  - Ice-breaker activity
  - Overview of the AI/ML landscape in Vietnam

### 9:00 – 10:30 AM | AWS AI/ML Services Overview
- **Focus:** Amazon SageMaker – End-to-end ML platform
- **Content:**
  - Data preparation and labeling
  - Model training, tuning, and deployment
  - Integrated MLOps capabilities
  - **Live Demo:** SageMaker Studio walkthrough

### 10:30 – 10:45 AM | Coffee Break
- **Content:** Networking break

### 10:45 AM – 12:00 PM | Generative AI with Amazon Bedrock
- **Content:**
  - **Foundation Models:** Claude, Llama, Titan – comparison & selection guide
  - **Prompt Engineering:** Techniques, Chain-of-Thought reasoning, Few-shot learning
  - **Retrieval-Augmented Generation (RAG):** Architecture & Knowledge Base integration
  - **Bedrock Agents:** Multi-step workflows and tool integrations
  - **Guardrails:** Safety and content filtering
  - **Live Demo:** Building a Generative AI chatbot using Bedrock

## Outcomes and Value Gained

### Technical Knowledge
- **SageMaker Platform:** Comprehensive understanding of end-to-end ML platform capabilities including data preparation, model training, and MLOps
- **Foundation Models:** Deep knowledge of different foundation models (Claude, Llama, Titan) and selection criteria
- **Prompt Engineering:** Practical skills in prompt engineering techniques, Chain-of-Thought reasoning, and Few-shot learning
- **RAG Architecture:** Understanding of Retrieval-Augmented Generation implementation and Knowledge Base integration
- **AI Safety:** Knowledge of implementing guardrails and content filtering in AI applications

### Practical Skills
- **SageMaker Studio:** Hands-on experience with SageMaker Studio interface and workflows
- **Bedrock Implementation:** Practical experience building generative AI chatbots using Amazon Bedrock
- **MLOps:** Understanding of integrated MLOps capabilities and best practices

### Professional Development
- **AI/ML Landscape:** Comprehensive overview of AI/ML opportunities and trends in Vietnam
- **Networking:** Connected with AI/ML practitioners and AWS community members
- **Best Practices:** Learned industry best practices for AI/ML implementation on AWS

This workshop provided essential foundation knowledge for implementing AI/ML solutions on AWS, with particular focus on generative AI capabilities and practical hands-on experience with key AWS services.
